## Installation

### APP (ReactJS)
1. Clone the repository:

   ```bash
   git clone https://github.com/owlCoder/oib-ui.git
   ```

2. Navigate to the project directory:

   ```bash
   cd oib-ui
   ```

3. Install dependencies:

   ```bash
   npm install
   ```
